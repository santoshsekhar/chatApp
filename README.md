# Requirements

- Node JS
- Git
- IDE (Visual Studio Code etc.,)

# How to run

- clone repo using "git clone https://github.com/santoshsekhar/chatApp"
- Open project folder in IDE
- in terminal type "npm install"
- to start "node index.js or nodemon"
